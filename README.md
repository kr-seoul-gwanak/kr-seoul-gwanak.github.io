# 관악구 · 서울특별시

> 관악구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-gwanak.github.io/**

관악구는 관악산 자락에 안겨 있는 대학과 청년의 도시로, 서울대학교 관악캠퍼스와 신림동 고시촌이라는 두 상징으로 널리 알려져 있습니다. 구 이름부터 갓(冠)을 세워 놓은 듯한 큰 산(岳)이라는 관악산의 형세에서 비롯되어, 지명 자체가 뒷산의 모습을 그대로 담고 있습니다. 고려의 명장 강감찬이 태어난 낙성대가 이곳에 있어 호국의 고장이라는 자부심도 함께 이어져 오고 있습니다. 봉천동과 신림동을 축으로 하는 밀집된 주거지에는 원룸과 고시원, 다세대주택이 촘촘히 들어차 있어 젊은 세대의 첫 서울살이 관문 역할을 해 왔습니다. 서울대입구역과 신림역 상권, 순대타운, 녹두거리 같은 생활 문화 공간이 어우러져 활기찬 도시의 표정을 만들어 냅니다. 학문과 서민의 삶, 그리고 산과 하천의 자연이 한데 어우러진 것이 관악구만의 개성입니다.
